# IT Help Desk Management Portal

## Project Overview

Developed a comprehensive IT Help Desk Management Portal on ServiceNow to streamline incident management, service requests, knowledge management, and SLA tracking. The portal enables employees to raise requests, access support resources, and track ticket status through a self-service interface.

## Features

* Incident Management with automated assignment
* Service Catalog for Laptop, Software, and VPN requests
* Knowledge Base with self-help articles
* Flow Designer automations for request processing
* SLA Management and breach escalation
* Service Portal for employee self-service
* Dashboard reporting and analytics

## Technologies Used

* ServiceNow
* Flow Designer
* Service Catalog
* Knowledge Management
* SLA Management
* Service Portal
* Reports and Dashboards

## Automation Workflows

1. Auto Assign Incident by Category
2. Incident Resolved Notification
3. Catalog Request Notification
4. Catalog Request Fulfillment
5. SLA Breach Escalation
